Alert Setup

Use native alerts first, then a free monitor for pages that support stable watch areas.

  1. Open a route from this tracker and turn on Google Flights price tracking for that exact itinerary or route.
  2. Open Skiplagged, sign in, and use its Create a Price Alert control for the same route.
  3. Use ITA Matrix as the verification source. It is strong for advanced search syntax, but it does not provide native public price alerts.
  4. For a Chrome-based workflow, install Distill Web Monitor, create a monitor from a results page, select the fare area, set the interval to 48 hours, and add an email or desktop notification action.
  5. When an alert fires, log the best fare here. The app flags target hits and keeps a history by source.
